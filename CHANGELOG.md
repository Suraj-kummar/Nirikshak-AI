## [1.0.0] - 2024-06-18

### Added
- React 18 frontend with Material UI v5 design system
- Spring Boot 3.x REST API with JWT authentication
- Python FastAPI AI vision service with MediaPipe
- Real-time gaze tracking and head pose detection
- Face detection and multiple-face alerts
- WebSocket live proctoring feed
- Violation logging and exam session management
- PostgreSQL schema with Flyway migrations
- Docker Compose orchestration for all services
- Nginx reverse proxy configuration
- Vercel deployment config for frontend
- Render.com blueprint for backend/AI deployment
- JUnit 5 unit tests for backend services
- PyTest setup for AI vision service
- Comprehensive README with installation guide
- API reference documentation
- Environment variable templates

### Security
- Stateless JWT-based authentication
- Bcrypt password hashing
- Role-based access control (ADMIN / CANDIDATE)
- CORS configuration for production domains
